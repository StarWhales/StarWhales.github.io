发表于2025-07-15|更新于2025-07-16

1.某公司内网网络被黑客渗透，请分析流量，给出黑客使用的扫描器

  • http contains "扫描器关键词"

    • acunetix — AWVS

    • sqlmap（sql注入常在user-agent字段注入，排查语法：http.user_agent contains "sqlmap"）

    • nmap （扫描结果为空）

    • nessus （扫描结果为空）

    • Appscan（扫描结果为空）

2.某公司内网网络被黑客渗透，请分析流量，得到黑客扫描到的登陆后台是(相对路径即可)

登录后台：/admin/login.php

排查语句：http contains "login" && http.request.method=="POST"

  • 登录请求通常为POST
  • 扫描关键词
    • login
    • admin
    • manager
    • system

3.某公司内网网络被黑客渗透，请分析流量，得到黑客使用了什么账号密码登陆了web后台(形#式:username/password)

两组账号密码：

  1. user_name=admin&password=admin%21%40%23pass123

  2. user_name=%E4%BA%BA%E4%BA%8B&password=hr123456

  3. 排查语句：

    • http contains "login" && http.request.method=="POST"后找到最底下的账户密码，追踪流查看是否302跳转

    • http.response.code==302后查看最底下的流量，找到Request URI字段为**/admin/login.php?rec=login**的流量，追踪流查看请求的账号密码
