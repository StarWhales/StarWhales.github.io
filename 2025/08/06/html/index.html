各种语言的语法知识

发表于2025-08-06|更新于2025-10-26
|浏览量:

—HTML：

标签：

说明 表示 单双标签
标题标签 h1-h6 双标签
段落标签 p 双标签
超链接标签 a 双标签
表单标签 form 双标签
表单元素标签 input 单标签
图片标签 img 单标签
横线标签 hr 单标签

部分特殊字符：

特殊字符 实体字符（原样显示）
空格 &nbsp;
小于号 &lt;
大于号 &gt;
引号 &quot;
版权符 &copy;

form表单：

属性 值 描述
action url 设定表单数据提交到的后端接口地址（如 PHP、Java 等处理页面）
method get、post 设定表单数据的提交方式： - get：数据通过 URL 参数传递，适合简单、非敏感数据 - post：数据通过请求体传递，更安全，适合敏感数据（如密码）或大量数据
enctype multipart/form-data 当表单包含 <input type="file"> 等文件上传控件时，必须使用该值，否则文件无法正常上传

input标签：

属性 可能值 描述
type text、password、radio、checkbox、submit、button、reset、image、file、hidden 设定 input 元素的类型（决定元素的外观和行为）
name name（自定义名称） 设定元素的名称，用于表单提交时标识数据
value value（自定义值） 设定元素的初始值或提交值
maxlength number（数字） 设定输入框可输入的最大字符长度（适用于 text、password 等类型）
type 属性值 具体描述
text 单行文本输入框，用于输入普通文本
password 密码输入框，输入内容会以圆点或星号隐藏
radio 单选框，多个同 name 的单选框互斥（只能选一个）
checkbox 复选框，多个同 name 的复选框可多选
submit 提交按钮，点击后提交表单数据到 action 地址
button 普通按钮，需配合 JavaScript 实现自定义功能
reset 重置按钮，点击后将表单元素恢复到初始状态
image 图片按钮，功能类似 submit，用图片作为按钮外观
file 文件选择控件，用于上传文件（需配合 form 的 enctype 属性）
hidden 隐藏域，不在页面显示，用于存储表单的隐藏数据并随表单提交

—JavaScript：

undefined 类型只有一种值，就是 undefined。一般是申明变量没有初始化就为 undefined。

null 数据类型的值也只有一个，就是 null，null 代表为空，null的数据类型为object。

console.log(i);：在控制台打印i的值

==与===：前者判断值是否相等，后者还需判断类型是否相同（是否全等）

var day = parseInt(prompt('请输入星期几：', '1'));：prompt获取用户输入，后一个引号中的内容会出现在浏览器弹窗的输入框内，parseInt将字符串转换为整数

事件：

  • onload：加载完成时执行。写在body标签 onload = “init()” 与单独使用 window.onload = function{}; 效果相同，在img标签时监听单张图片加载完成
  • onresize：当页面的尺寸发生变化时所执行的代码
  • onfocus：获取焦点时触发
  • onblur：失去焦点时触发

—PHP

  • strlen($str1);：计算字符串长度

  • strrev($str);：反转字符串

  • strtoupper()：将字符串全部改写为大写形式；

  • strtolower()：将字符串全部改写为小写形式。

  • strpos($str,'world');：查找子字符串，返回第一个匹配的字符位置（从0开始）

  • echo str_replace($str2, $str3, $str1);：将$str1中的$str2用$str3替换

     // 声明属性

     }

// 实例化对象

  • $ext = substr(strrchr($_FILES['upload']['name'], '.'), 1);：#获取名为upload的文件后缀，其中.为0，后缀第一个字符为1，从1开始截取

  • 会话管理和控制：

    • //Cookie是一种在客户端（用户的计算机）上存储数据的方式，可以用于跟踪用户会话、存储用户偏好设置等。
// 设置cookie 
setcookie('name','ryan',time()+3600); 
// 打印cookie 
print_r($_COOKIE); 
echo "<br>"; 
echo $_COOKIE['name']; 
// 删除cookie 
setcookie('name','ryan',time()-3600);

Python

  • sorted(可迭代对象, reverse=False) ：一个内置函数，用于对可迭代对象（如列表、元组、字符串等）进行排序，并返回一个新的排序后的列表，不会修改原对象。要让 sorted() 函数按降序排序，只需指定其 reverse 参数为 True 即可（默认是 False，表示升序）
  • a, b = b, a：右侧的 b, a 会被自动打包成一个元组 (b, a)（临时创建）
  • 包含可变元素的元组：
  • selected_courses = {'Math', 'Python', 'AI', 'Math'} # 自动去重
  • electives = {f"E{num:02d}" for num in range(1,6) if num%2}：因为创建的集合，元素顺序随机
